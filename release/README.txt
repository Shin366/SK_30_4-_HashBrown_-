HashBrown · AI 포렌식 분석 대시보드 — 단독 실행 파일
=====================================================

[ 사용법 ]
  HashBrown.exe 를 더블클릭하면 대시보드가 앱 창으로 열립니다.
  업로드 화면에서 로그/MFT 파일을 끌어다 놓거나, 데모용 합성 로그
  카드를 눌러 바로 체험할 수 있습니다.

[ 배포 ]
  이 파일(HashBrown.exe) 하나만 보내면 됩니다. 앱(HTML/JS/CSS)이
  exe 안에 모두 들어 있어, 받는 사람은 설치할 것이 전혀 없습니다.

[ 동작 방식 ]
  실행하면 내장된 앱을 %LOCALAPPDATA%\HashBrown\ 에 풀어놓고,
  Microsoft Edge 또는 Chrome 을 앱(window) 모드로 띄웁니다.
  (둘 다 없으면 기본 브라우저로 열립니다.)

[ 요구 사항 ]
  - Windows (.NET Framework 4.x — 모든 최신 Windows 에 기본 포함)
  - Edge 또는 Chrome (Edge 는 Windows 에 기본 탑재)

[ 처음 실행 시 경고가 뜨면 ]
  서명되지 않은 프로그램이라 Windows SmartScreen 이
  "Windows의 PC 보호" 경고를 띄울 수 있습니다.
  → "추가 정보" → "실행" 을 누르면 됩니다.
  (회사 백신이 차단하면 예외 등록이 필요할 수 있습니다.)

[ 개인정보/보안 ]
  모든 분석은 브라우저 내부(로컬)에서만 수행되며 데이터가
  외부로 전송되지 않습니다. 인터넷 없이도 동작합니다.
  (글꼴만 온라인일 때 더 예쁘게 표시되고, 오프라인이면 시스템
   기본 글꼴로 대체됩니다.)
